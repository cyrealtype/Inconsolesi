FONTLOG for the Inconsolesi fonts

This file provides detailed information on the Inconsolesi
Font Software.

This information should be distributed along with the 
Inconsolesi fonts and any derivative works.

Basic Font Information

Alexei Vanyashin, the primary designer of the family, wrote:

Inconsolesi is a monospaced design based on 
Inconsolata by Raph Levien(firstname.lastname@gmail.com) with 
full WLG-4 support.

Inconsolesi is a Unicode typeface family that supports 
languages that use the Latin script and its variants, 
and could be expanded to support other scripts.

More specifically, this release supports the following 
Unicode ranges: WGL-4

Source files

There are four FontLab merged and optimized files for each weight:

Inconsesi-Bold-OTF.vfb
Inconsesi-BoldItalic-OTF.vfb
Inconsesi-Regular-OTF.vfb
Inconsesi-RegularItalic-OTF.vfb
Inconsolesi-1003.fea (Opentype features file)
Inconsolesi-1003.flc (Classes file)
